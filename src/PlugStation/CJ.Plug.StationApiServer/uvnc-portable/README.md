# UltraVNC Portable Files

将以下文件放入此目录:

1. `winvnc.exe`    — UltraVNC 服务端主程序
2. `VNCHooks.dll`  — UltraVNC 钩子库

## 下载方式

从 UltraVNC 官网下载 UltraVNC_X64 的 zip 版本:
- 官网: https://www.uvnc.com/downloads/ultravnc.html
- 从 zip 中提取 `winvnc.exe` 和 `VNCHooks.dll` 即可

注意: SCHook.dll 在新版 UltraVNC 中已不再单独提供，不需要此文件。
